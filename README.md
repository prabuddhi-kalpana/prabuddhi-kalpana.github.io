# Simplified Academic Website

This is a simplified version of the academicpages template, focused on research and blog content with improved typography and a cleaner design.

## Features

- Streamlined navigation with focus on research and blog content
- Improved typography using Google Fonts (Open Sans for body text, Roboto Slab for headings)
- Modern, card-based design for content items
- Mobile-responsive layout
- Simplified configuration

## Structure

- `_config.yml` - Main configuration file (simplified)
- `_data/navigation.yml` - Navigation menu structure
- `_includes/` - Template components
- `_layouts/` - Page layouts
- `_pages/` - Static pages (about, cv, etc.)
- `_posts/` - Blog posts
- `_publications/` - Research publications
- `_sass/` - Styling components
- `assets/` - CSS, JavaScript, and other assets
- `files/` - Downloadable files
- `images/` - Images for the site

## Customization

1. Update `_config.yml` with your personal information
2. Replace `images/profile.png` with your profile photo
3. Edit pages in `_pages/` to update your content
4. Add blog posts to `_posts/` and publications to `_publications/`

## Removed Components

The following components from the original academicpages template were removed to simplify the site:

- Portfolio section
- Talks section
- Teaching section
- Talkmap visualization
- Markdown generator tools

## Future Updates

To further customize your site:

1. Modify `assets/css/main.scss` to change colors and styling
2. Edit layout files in `_layouts/` for structural changes
3. Update navigation in `_data/navigation.yml` to add or remove sections
