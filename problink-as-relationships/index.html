This dataset contains AS relationships inferred on the first day of every month starting from 2019 using
ProbLink described in "Stable and Practical AS Relationship Inference with ProbLink" in NSDI'19.

Data Format:
<provider-as>|<customer-as>|-1
<peer-as>|<peer-as>|0
<sibling-as>|<sibling-as>|1
